# KWebView
An easy to use kotlin-js library for client-side frontend development.

1. [Quick start](#quick-start)

## Quick start
To use this library simply add this to your `build.gradle`
```groovy
repositories {
    jcenter()
}

dependencies {
    implementation "de.westermann:KWebView:$version"
}
```
